# 三国群英传2 素材需求清单

## 角色头像
- 尺寸: 64x64 像素
- 格式: PNG 带透明通道
- 数量: 400+ 武将
- 风格: 像素风格或低多边形风格

## 地图素材
- 中国地图底图
- 城池图标 (正常/选中/占领状态)
- 道路连接线
- 地形图标 (平原/山地/水城/沙漠)

## 战斗素材
- 武将战斗立绘
- 兵种图标 (步兵/骑兵/弓兵)
- 技能特效动画
- 伤害数字

## UI素材
- 按钮样式
- 面板边框
- 血条/蓝条
- 字体 (复古风格)

## 音效音乐
- 背景音乐 (各场景)
- 战斗音效
- 技能音效
- UI音效

## 数据来源参考
1. 原版游戏截图参考
2. 网络资料整理
3. 自制或AI生成

## 素材目录结构
```
data/
├── characters/
│   ├── avatars/      # 头像
│   ├── portraits/    # 立绘
│   └── sprites/      # 战斗精灵
├── maps/
│   ├── tiles/        # 地图瓦片
│   ├── cities/       # 城池图标
│   └── backgrounds/  # 背景图
├── skills/
│   ├── icons/        # 技能图标
│   └── effects/      # 技能特效
└── music/
    ├── bgm/          # 背景音乐
    └── se/           # 音效
```
