# 素材目录

## 武将头像 (characters/avatars/)
命名格式: {武将ID}.png
尺寸: 64x64 像素

## 武将立绘 (characters/portraits/)
命名格式: {武将ID}_full.png
尺寸: 256x256 像素

## 地图素材 (maps/)
- tiles/: 地图瓦片
- cities/: 城池图标
- china_base.jpg: 地图底图

## UI素材 (ui/)
- buttons/: 按钮样式
- panel/: 面板背景
- icons/: 小图标

## 技能图标 (skills/)
命名格式: skill_{技能ID}.png
尺寸: 32x32 像素
