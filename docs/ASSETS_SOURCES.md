## 七、下载脚本

### 批量下载Wikimedia图片

```bash
#!/bin/bash
# download_avatar.sh
cd ~/sanguo-hero/assets/characters/avatars

# 模板URL (替换武将名)
URL="https://upload.wikimedia.org/wikipedia/commons/thumb/X/XX/Name_Qing_portrait.jpg/220px-Name_Qing_portrait.jpg"

# 下载函数
download() {
  local name=$1
  local filename=$2
  curl -sL "$url" -o "$filename" --max-time 10
  if [ -s "$filename" ] && [ $(stat -f%z "$filename") -gt 1000 ]; then
    echo "✓ Downloaded: $filename"
  else
    rm -f "$filename"
    echo "✗ Failed: $filename"
  fi
}

# 批量下载
for hero in "蒋琬" "费祎" "董允"; do
  download "$hero" "$(echo $hero | pinyin).jpg"
done
```

## 八、版权说明

### 安全使用的素材类型
1. ✅ **Public Domain (公有领域)** - 版权已过期或放弃
2. ✅ **CC0** - 作者放弃版权
3. ✅ **CC-BY** - 需要署名
4. ✅ **专有授权** - 购买或获得授权

### 避免使用的素材
1. ❌ 未经授权的商业游戏截图
2. ❌ 仅供个人使用的素材用于商业项目
3. ❌ 来源不明的网络图片

### 建议使用
- Wikimedia Commons 公有领域图片
- CC0许可的游戏素材
- 购买正版素材包
- 自行创作
