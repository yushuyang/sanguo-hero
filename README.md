# 三国群英传2 复刻项目

## 项目概述
基于三国群英传2的浏览器复刻版，保留原作核心玩法，使用现代Web技术实现。

## 游戏特色
- **五个历史时期**：黄巾之乱、讨伐董卓、群雄割据、赤壁之战、三国鼎立
- **400+武将**：完整武将数据，包含武力、智力、武器、技能
- **即时战略**：非回合制，时间以月计算
- **战争主导**：重点是战争而非内政

## 项目结构

```
sanguo-hero/
├── data/
│   ├── characters/     # 武将数据
│   ├── maps/          # 地图数据
│   ├── scripts/       # 剧本配置
│   ├── skills/        # 技能数据
│   ├── items/         # 武器道具
│   ├── music/         # 音效音乐
│   └── sprites/       # 精灵图素材
├── src/
│   ├── game/          # 游戏核心逻辑
│   ├── ui/            # 界面组件
│   └── utils/         # 工具函数
├── assets/            # 静态资源
└── docs/              # 文档资料
```

## 数据来源
- 百度百科 - 三国群英传II
- 游戏原始数据整理
- 社区玩家资料汇总
